# 🏏 IPL Match Prediction Project

This project is a Machine Learning-based web application that predicts the outcome of IPL matches using historical data. It also includes exploratory data analysis (EDA) and an interactive user interface built with Streamlit.

---

## 🚀 Features

* 📊 Data Analysis of IPL dataset
* 🤖 Machine Learning model for match prediction
* 🌐 Interactive web app using Streamlit
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* Python
* Pandas & NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Streamlit

---

## 📁 Project Structure

```
ipl-project/

├── data/              # Dataset files
├── notebook/          # EDA notebooks
├── model/             # Trained ML model (.pkl)
├── app.py             # Streamlit app
├── requirements.txt   # Dependencies
└── README.md
```

---

## ⚙️ Installation & Run

1. Clone the repository:

```
git clone https://github.com/your-username/ipl-prediction-project.git
```

2. Navigate to project folder:

```
cd ipl-prediction-project
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
